# DiaryHub Fullstack Project

This project consists of a Django backend (server) and a React/Vite frontend (client).

## Structure
- `server/`: Django REST Framework backend.
- `client/`: React/Vite frontend.

## Getting Started

### Server
1. `cd server`
2. `source venv/bin/activate`
3. `python manage.py runserver`

### Client
1. `cd client`
2. `npm install`
3. `npm run dev`

## Environment Variables
- `server/.env`: Contains Django settings and superuser credentials.
- `client/.env`: Contains `VITE_API_URL`.
